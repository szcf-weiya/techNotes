    配置nginx¶

    很简单，只需要 

    sudo apt-get install nginx

    接着在浏览器中输入网址，便可以看到Welcome界面。

    参考 1. 基于阿里云搭建微信小程序服务器(HTTPS) 2. 在阿里云 ECS 搭建 nginx https nodejs 环境 (一、 nginx)
